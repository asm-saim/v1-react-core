##JSX:
JSX is a syntax extension for js.
JSX stand for js + xml.
We can write HTML-like markup inside jsx.
multiple line markup text must be in a fragmen or in a tag(div).
attribute must be used camel case if more than 2 word.ex: textAlign
all tags must be a closing tag.<></>
for dynamic value , in jsx we will use {} not ${}


##There are 3 types to express css style in react:
1. in .css file, then push it in className.
2. create a object in jsx, then call it in component as style={obj name} 
3. directly implement the style. style={{color:"red"}}
